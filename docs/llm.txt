# PDF Modifier MCP - LLM Context

## Overview
PDF text modification tool with dual interfaces: CLI (humans) and MCP server (AI agents).

## Installation
```bash
pip install pdf-modifier-mcp
```

## MCP Tools (for AI agents)
- `read_pdf_structure(input_path)` - Extract PDF structure as JSON
- `inspect_pdf_fonts(input_path, terms)` - Find text and report font properties
- `modify_pdf_content(input_path, output_path, replacements, use_regex=False)` - Replace text

## Replacement Syntax
- Simple: `{"old text": "new text"}`
- Regex: `{"\\d{4}": "YEAR"}` with `use_regex=True`
- Hyperlink: `{"Click": "Visit|https://example.com"}`
- Neutralize link: `{"Click": "Click|void(0)"}`

## CLI Commands
```bash
pdf-mod modify input.pdf output.pdf -r "old=new"
pdf-mod modify input.pdf output.pdf -r "pattern=replacement" --regex
pdf-mod analyze input.pdf --json
pdf-mod inspect input.pdf "search term"
```

## Key Behaviors
- Preserves font style (maps to Base 14 fonts)
- Redacts original text with white fill before inserting replacement
- Replacement text should be similar length to avoid overlap
- Returns JSON with success status, replacement count, warnings
